# Runbook conventions

Runbooks are one half of the alerts + runbooks definition-of-done for new
indexers. The canonical guidance (what to cover, how to pair each runbook
`## AlertName` section with its alert rule) lives in
[../../alerts/AGENTS.md](../../alerts/AGENTS.md). Read and apply that file before
modifying a runbook; it is in a sibling subtree and is not loaded automatically.
